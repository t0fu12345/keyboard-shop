# TACTILE LUXE - Keyboard Shop (Practice Project)

Welcome to **TACTILE LUXE**, a mechanical keyboard shop interface built with React and Tailwind CSS. 

> **Note:** This website is created solely for **practicing frontend development and UI design**. It is not a real e-commerce store, and no actual purchases can be made.

## View the Live Site (GitHub Pages)

This project is automatically deployed to GitHub Pages using GitHub Actions. You can view the live demo anytime at the following link:

**[https://t0fu12345.github.io/keyboard-shop/](https://t0fu12345.github.io/keyboard-shop/)**

*(Note: This website is hosted 24/7 on GitHub Pages and is accessible to anyone on the internet.)*

## Local Development

If you want to run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/t0fu12345/keyboard-shop.git
   ```
2. Navigate into the project directory:
   ```bash
   cd "selling keyboard/keyboard-shop"
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open `http://localhost:5173` in your browser.
